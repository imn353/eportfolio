Semester II

Four courses — assignments, labs and capstone projects, each linking to its folder on GitHub.

SECI1143-02

Probability & Statistical Data Analysis

Throughout this course, two data-analysis projects were completed using R. In Project 1, survey data on exercise habits was collected and visualised, and inferential statistics were applied to assess physical and mental impacts; the findings were documented in a formal report. In Project 2, a public Life-Expectancy dataset was modelled, key relationships were tested, and the results were presented through R-generated slides. Probability distributions, hypothesis testing and exploratory data analysis were practised end-to-end, while reproducible R workflows were strengthened. Confidence in turning raw data into evidence-based conclusions has been visibly developed.

  • Project 1 — Exercise Habits and Its Impacts Survey-based R study with charts, an IEEE-style report and a project proposal.

  • Project 2 — Life Expectancy Analysis R analysis on a global Life-Expectancy CSV, presented with R slides.

SECJ1023-03

Programming Technique II

Across five C++ assignments and a group project, object-oriented programming was practised progressively. In the assignments, problems involving classes, inheritance, pointers, file I/O and the STL were solved one after another. In the group project, an initial storyboard was first produced and then translated by team S03 Group 6 into a working executable, with the codebase organised into clear source and header modules. Debugging, header/source separation and collaborative version control were reinforced throughout. The transition from procedural C++ to designing reusable, modular components has been internalised.

SECR1033-03

Computer Organization and Architecture

Across three labs and a final project, the inner workings of modern processors were explored at the assembly level. Lab exercises covered basic assembly, arithmetic operations and a recorded walkthrough of the lab outcomes. For the project, an x86 assembly program was written, benchmarked on a Lenovo 81T0 with Geekbench and discussed in a written report. Register usage, addressing modes and the cost of low-level operations were observed first-hand. A clearer mental model of how high-level code maps to hardware has been established.

SECJ2013-03

Data Structure and Algorithm

Two assignments, four labs and a mini project were undertaken to build fluency in classical data structures. Linked lists, stacks, queues and student-record management problems were implemented in C++ with custom header files. The mini project culminated in a flight-simulation program (v1.0 and v2.0) in which scheduling and search logic were applied. Algorithmic complexity, pointer manipulation and modular code organisation were practised throughout. The ability to choose an appropriate data structure for a given problem has been noticeably sharpened.

SECP3723-01

System Development Technology

A full PHP-based Course Registration System (crs_iman) was developed to consolidate practical web-development skills. Role-aware modules for administrators, lecturers and students were implemented, with screens for course management, registration approval and rejection, and detailed course views. Server-side scripting, session handling, form validation and navigation patterns were applied across the codebase. Multi-role authentication flows and CRUD-driven UI design were practised at scale. Confidence in delivering an end-to-end database-backed web application has been substantially strengthened.
