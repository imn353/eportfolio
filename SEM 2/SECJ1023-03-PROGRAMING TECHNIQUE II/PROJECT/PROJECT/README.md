## Description
The project is about designing a game using the object oriented approach in C++ programming that incorporates all the components in object-oriented programming in C++. The game needs to have an array of objects, the encapsulation concept, assosiation concept such as aggregation and composition and methods that are polymorphic in nature. We have designed a shooter based game that delivers all of the components needed for this project. 

## Gameplay

The game starts with the player at the bottom center of the screen. The player can be moved using the 'A' and 'D' keys where 'A' move the player to the left and 'D' key move the player to the right. To complete the game, the player must hit all the shapes in generated on the screen which the rectangkes and triangles and the player will be considered as the winner. The also player needs to avoid the bombs (circular shape) or the game ends and the player loses. To the the player, the 'W' key needs to be pressed to launch the player to the list of generated shapes. 


## Game Mission
The mission of the game is to complete the game by clearing all the shapes by hitting them with the player and avoiding the bombs or the game ends and the player loses. Clearing all the shapes will consider the player to be winners.


## Project Structure
The project structure consists of the the hittable objects file (shapes, triangles, rectangle, bomb files) where each of them consists of a cpp and hpp files. Other than that is the player file that consists of the player.hpp and player.cpp file. lastly is the game file where the game.cpp and game.hpp is located and this file facilitates the flow of gampeplay controlling when the game starts, running and when the game ends.
