Rainfall Data Analysis Assignment (100 Marks, Due 6th July 2025)¶

You are provided with a rainfall dataset recording monthly measurements from various weather stations across different regions. The dataset includes rainfall (mm), number of rainy days, maximum and minimum temperatures, and whether the station is in an urban or rural area.

Dataset columns:

  • Region: Name of the region (e.g., North, South, East, West)
  • Year: Year of measurement
  • Month: Month (e.g., Jan, Feb, ..., Dec)
  • Station: Weather station name/ID (e.g., StationA, StationB)
  • Rainfall_mm: Rainfall amount in millimeters
  • RainyDays: Number of days in the month with rain
  • MaxTemp_C: Maximum temperature recorded in that month (°C)
  • MinTemp_C: Minimum temperature recorded in that month (°C)
  • UrbanRural: "Urban" or "Rural"

# --- Annotating Highest and Lowest Points using ax.text() ---

# Find the overall max and min rainfall values and their corresponding indices
max_val = average_rainfall_by_station_month.max().max()
min_val = average_rainfall_by_station_month.min().min()

# Get the location (Station and Month) for the highest value
max_month_series = average_rainfall_by_station_month.max()
max_month = max_month_series.idxmax()
max_station_series = average_rainfall_by_station_month[max_month]
max_station = max_station_series.idxmax()

# Get the location (Station and Month) for the lowest value
min_month_series = average_rainfall_by_station_month.min()
min_month = min_month_series.idxmin()
min_station_series = average_rainfall_by_station_month[min_month]
min_station = min_station_series.idxmin()

# Get the index of the highest/lowest station on the x-axis
x_pos_max_station = list(average_rainfall_by_station_month.index).index(max_station)
x_pos_min_station = list(average_rainfall_by_station_month.index).index(min_station)

# Iterate over the bar containers to find the specific bar for annotation
for container in ax.containers:
    month_name = container.get_label()
    if month_name == max_month:
        for bar in container:
            if bar.get_x() + bar.get_width()/2 > x_pos_max_station - 0.5 and bar.get_x() + bar.get_width()/2 < x_pos_max_station + 0.5:
                if bar.get_height() == max_val:
                    max_bar_x = bar.get_x() + bar.get_width() / 2
                    max_bar_y = bar.get_height()
                    ax.text(max_bar_x, max_bar_y + 5, # Slightly above the bar
                            f'{max_val:.1f}',
                            ha='center', va='bottom',
                            fontsize=9, color='red', fontweight='bold',
                            bbox=dict(boxstyle='round,pad=0.2', fc='yellow', ec='red', lw=0.5, alpha=0.7)) # Optional bounding box

import matplotlib.dates as mdates

# Create a 'Date' column by combining 'Year' and 'Month'
month_mapping = {
    'Jan': 1, 'Feb': 2, 'Mar': 3, 'Apr': 4, 'May': 5, 'Jun': 6,
    'Jul': 7, 'Aug': 8, 'Sep': 9, 'Oct': 10, 'Nov': 11, 'Dec': 12
}
rain['Month_Num'] = rain['Month'].map(month_mapping)
rain['Date'] = pd.to_datetime(rain['Year'].astype(str) + '-' + rain['Month_Num'].astype(str) + '-01')

# Group by Date, Region, and UrbanRural and calculate the mean rainfall
df_grouped = rain.groupby(['Date', 'Region', 'UrbanRural'])['Rainfall_mm'].mean().reset_index()

# Create the facet/grid plot using seaborn.relplot
# We use relplot with kind='line' to show trends over time

    data=df_grouped,
    x='Date',
    y='Rainfall_mm',
    col='UrbanRural',
